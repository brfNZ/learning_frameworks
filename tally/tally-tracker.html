ⓘ Quick Reference

Tally Types

🔢Count — tap the big number to count up.
📈Progress — tap = win, hold = loss. Small red L shown alongside wins.
⚔️Win/Loss — tap either coloured block to log. Block widths reflect the win/loss ratio.
🎯Challenge — deadline only, no target. Tap count to log. Ends when deadline passes.

Target & Loss Limit

🎯Enter target as 10 for wins only, or 10/3 to set a loss limit of 3. Default loss limit is target÷2.

▶ Review History

😊Tap 😊 to open an emoji picker for standalone notes. Tap an emoji on any count entry to rate it. Pin notes after adding via the 📌 button in the entry.

🔥 Streak

🔥Consecutive calendar days logged. 🔥 3d = 3 days in a row. Miss a day and it resets.

📌 Pin

📌Click to pin a tally to the top. Ctrl+click pins with hot pink glow.

12W 3L

🃏Collapsed card summary. W = wins, L = losses. Also shows % and pace.

📅 Deadline

📅Set when adding or editing. Green = time to spare, amber = 7 days or less, red = overdue. Pace line appears above the bar.

💾 Backup & Restore

💾Export to file — full JSON backup of all tallies.
📋Copy JSON — copies all data to clipboard.
📂Import from file — opens a picker to choose which tallies to restore. All append to existing data.
📋Paste JSON — paste a backup, then pick which tallies to restore from the list.
